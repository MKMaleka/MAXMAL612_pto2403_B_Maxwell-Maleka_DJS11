# Podcast App

Welcome to the Podcast App! This application allows you to browse, search, and manage your favorite podcasts. You can explore shows, listen to episodes, and save your favorites for easy access.

- [Contact Information](#contact-information)

## Introduction

The Podcast App is a React-based web application designed to provide a user-friendly interface for discovering and enjoying podcasts. Key features include:

- **Show Discovery:** Browse a curated list of podcasts.
- **Detailed Show Information:** View show details, seasons, and episodes.
- **Audio Playback:** Listen to episodes directly in the app.
- **Favorites Management:** Save and organize your favorite episodes.
- **Search and Filtering:** Find shows by title, genre, or update date.
- **Responsive Design:** Enjoy the app on any device.

## Setup Instructions

To run the Podcast App locally, follow these steps:

1.  **Clone the Repository:**

    ```bash
    git clone [https://github.com/MKMaleka/MAXMAL612_pto2403_B_Maxwell-Maleka_DJS11.git]
    cd [MAXMAL612_pto2403_B_Maxwell-Maleka_DJS11]
    ```

2.  **Install Dependencies:**

    ```bash
    npm install react-router-dom
    ```

3.  **Start the Development Server:**

    ```bash
    npm run dev
    ```

    This will launch the app in your default browser at `http://localhost:5173/`.

4.  **Install Fuse.js for fuzzy search:**

    ```bash
    npm install fuse.js
    ```

## Usage Examples

### Browsing Shows

-   Navigate to the home page to view a list of available shows.
-   Use the search bar to find shows by title or description.
-   Filter shows by genre using the genre dropdown.
-   Sort shows alphabetically or by update date using the dropdown.
-   Click on a show to view detailed information.

### Listening to Episodes

-   On the show details page, select a season.
-   Click the "Play" button next to an episode to start playback.

### Managing Favorites

-   On the show details page, click "Add to Favorites" to save an episode.
-   Navigate to the "Favorites" page to view your saved episodes.
-   Sort your favorites by title or update date using the dropdown.
-   Click "Remove" to delete an episode from your favorites.
